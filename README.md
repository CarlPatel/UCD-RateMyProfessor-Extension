# üìò UC Davis Schedule Builder RateMyProfessors Extension

This Chrome extension enhances UC Davis Schedule Builder by displaying **RateMyProfessors ratings directly next to each course instructor** ‚Äî no more switching tabs or manually searching.

Hover over the rating badge to see:
- Professor name
- Rating
- Difficulty
- Would Take Again %
- Click for a quick link to their RMP profile  
- Google search fallback if no profile is found

---

## üì¶ Installation (From ZIP)

### Step 1 - Download the ZIP

Download the latest release ZIP file from this repository: 

[ucd-schedulebuilder-rmp.zip](https://github.com/CarlPatel/UCD-RateMyProfessor-Extension/releases/download/v1.0/ucd-rmp-extension.zip)

---

### Step 2 - Extract the ZIP

Unzip the file so you have the folder: `ucd-schedulebuilder-rmp/`

---

### Step 3 - Open Chrome Extensions Page

In Chrome, go to extensions manager:

```
chrome://extensions
```

---

### Step 4 - Enable Developer Mode

In the top-right corner, toggle **Developer Mode**

---

### Step 5 - Load the Extension

Click **Load unpacked**  
Select the extracted folder (not the ZIP)

---

### Step 6 - Done!

Go to [UC Davis Schedule Builder](https://my.ucdavis.edu/schedulebuilder/) and you should see RateMyProfessor ratings appear next to instructors automatically.

---
